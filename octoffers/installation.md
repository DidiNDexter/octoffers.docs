From source
------

**Before you will start ensure that you have this:**
- **python 3.9+**
- **chrome webdriver**
- **sqlite3**

**Follow this steps:**

1) **Install required dependency**
`pip install -r requirements.txt`
2) **Pull Private Drivers** (if you have access)
`git submodule update --init --recursive`
3) **Use OctOffers**
`python octoffers`
